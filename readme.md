# Aplikasi Pustaka Booking

### Apa itu aplikasi Pustaka Booking?

Aplikasi Pustaka Booking ini adalah aplikasi untuk perpustakaan dalam mengelola data peminjaman ataupun pengembalian buku

### Kelebihan Project
1. Meminjam buku
2. Mengembalikan buku
